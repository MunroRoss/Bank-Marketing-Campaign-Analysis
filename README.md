# Bank-Marketing-Campaign-Analysis
This project involved conducting exploratory data analysis and developing statistical models using R for a bank's telemarketing campaign to classify the outcome of a phone call and whether it results in a sale.

In this application, the aim is to develop statistical models that can accurately predict the result of a phone call to sell long term bank deposits. Such models are valuable as they assist campaigners to prioritize and select the next customers to be contacted during marketing campaigns. Improving the efficiency of such campaigns reduces time and associated cost.

The first section covers Exploratory Data Analysis, and the second part involves model training and evaluation. All analysis and training were done using R.
